# Portalist

**Portalist** is a lightweight travel menu for **World of Warcraft Retail**, giving you instant access to your most useful teleports through a single, fast dropdown.

Open it at your cursor with a keybind, and Portalist dynamically shows only the travel options you’ve enabled *and* can currently use.

---

## Supported Travel

* Challenge Mode Teleports
* Hearthstones
* Engineering Wormholes
* Mage Portals
* Mailboxes

---

## Features

* Single compact dropdown for all travel options
* Fully configurable list of spells, toys, and destinations
* Seamless support for both spells and toys in one menu
* Challenge Mode portals grouped by expansion
* Dual keybind support
* Simple slash commands for quick access
* Smart filtering — only shows usable and enabled entries

---

## Slash Commands

* `/portalist`
* `/port`

Opens the configuration window.
